# Poggle Mapper

Map Purchase Order numbers to General Ledger IDs.

## Windows

1. `git clone git@github.com:americanhanko/poggle-map.git`
1. `cd poggle-map`
1. `Invoke-PoggleMapper.ps1`

## Unix

1. `git clone git@github.com:americanhanko/poggle-map.git`
1. `cd poggle-map`
1. `pwsh Invoke-PoggleMapper.ps1`
